# 🚀 FocusCore - AI Destekli Üretkenlik Koçu

AI destekli odaklanma ve üretkenlik artırma platformu.

## 🛠️ Teknolojiler
- Python + Streamlit
- Google Gemini API
- SQLite Database
- LangChain

## 🚀 Kurulum
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 API Key
`.env` dosyasına `GOOGLE_API_KEY=your_key` ekleyin.
